## 一、查看已有索引及大小
```
1. 浏览器地址栏输入：
http://localhost:9200/_cat/indices?v&pretty

2.利用pyelasticsearch
import pyelasticsearch as pyes
from pprint import pprint

es = pyes.ElasticSearch(["http://localhost:9200"])
schema = es.get_mapping()
pprint(schema)
```
