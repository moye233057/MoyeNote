 dir()函数可以查看对象内的所有的属性和方法。
 在 python 中任何东西都是对象，一种数据类型，一个模块等，都有子集的属性和方法。
 除了常用的方法外，其他的你不需要全部记住它，交给 dir() 函数就好了。

 例子：
 一、查看一个列表有哪些方法。
 dir([])
